# Erebor: neue Questkarten

Ergänzt drei neue Questkarten für die geplante Kampagne „Orkische Überraschung“:

- Streit schlichten
- Kriegslager
- Ansturm der Belagerer

Die Kartendatensätze wurden als letzte Einträge in `OBJECTIVES` ergänzt. Die zugehörigen Effektblöcke stehen im Quest-Abschnitt von `EFFECTS` direkt hinter `AnsturmderHorde` und sind im bestehenden kompakten Einzeilenstil formatiert. Die Reihenfolge für `nameDeck` hängt ausschließlich von den Kartendatentabellen ab.

Skalierung:

- Streit schlichten: `w={13,15,18,20}`, `vw=2`
- Kriegslager: `w={24,30,36,40}`, `hw=2`, `vw=4`
- Ansturm der Belagerer: `w=2`, `vw=1`
