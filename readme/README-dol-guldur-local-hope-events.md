# Dol Guldur: lokale Hope-Events für Mission 3

Ergänzt in `missionDolGuldur3`:

- **Zusammenprall** (3 × Spielerzahl): Erschöpft bis zu zwei zufällige spielbereite Ork-Gegner.
- **Sie kommen nicht vorbei!** (6 × Spielerzahl): Wiederholt einmal pro Spieler; ein zufälliger Charakter erhält jeweils dauerhaft +2 Willenskraft.
- **Vorübergehende Barrikade** (9 × Spielerzahl): Alle derzeit spielbereiten Charaktere ohne Block erhalten Block bis zum Ende der Runde.

Geänderte Dateien:

- `LoTR-GameData.lua`
- `LoTR-Trans.lua`

`LoTR.lua` und `LoTR-CardData.lua` wurden nicht verändert.
